Solving CS:GO and Nitama Loader problems

1. Game crashes / DLC does not inject

1.1. Go to the settings, click on the "Virus and threat protection", then under the "Virus and threat protection settings" block you see the blue text "Manage settings", click on it and turn off "Real-time protection". After you turned off the antivirus, you must go back to the search. And write "Windows Defender Firewall", go there. Then in the left block "Turn Windows Defender Firewall on or off", click there. And check the boxes everywhere "Disable firewall ...". Restart the loader.

1.2. If this happened when you first started the loader, try installing the packages from here: Components. If you just installed them, try rebooting the system.

1.3. Clear CS:GO cache, check the game files in the properties. If the solutions above didn't help you: Disable multicore processing in the CS:GO game settings. Start DLC.

6. When downloading the update, it says - An unexpected error has occurred. / Loader file disapeared

Disable all antiviruses. Disable firewall.

7.How to open the cheat panel?
